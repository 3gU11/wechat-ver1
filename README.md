# 经销商订货门户小程序

这是面向中走丝线切割设备经销商的微信小程序前端框架。

它不是一套新的库存系统，而是现有 V7STD1.0 智能生产排产与成品库存管理系统的外部门户层。

## 当前能力

- 手机号提交普通经销商注册申请
- admin 审核经销商账号
- 审核通过后普通经销商才能登录
- 普通经销商查看机型、成品可订数量和在制可订数量
- 按预计入库时间和批次号查看对应机型数量
- 点击预计入库批次下的机型上传订单申请
- 查看经销商自己的订单申请

## 演示账号

```text
admin: admin / admin123
已审核经销商: 13800000000 / 123456
待审核经销商: 13900000000 / 123456
```

## 架构边界

```text
微信小程序前端
  -> 经销商专用 API
  -> 现有 FastAPI 业务中台
  -> 现有 MySQL 数据库
```

小程序不直接访问内部库存系统，不直接操作原始库存数据。
